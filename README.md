# AutoUser script for creating, removing, showing User

![image](./autouser.gif)


# Make the available to access as global, give permission
`chmod a+x autouser`
then
`sudo autouser /bin/`

## Creating User
`autouser -c user_name user_name`


## Deleting User
`autouser -r user_name user_name2`

## Show Users
`autouser -s`
